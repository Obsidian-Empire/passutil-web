# Passutil — генератор паспорта для Obsidian Empire

Инструмент для сервера **Obsidian Empire**, который позволяет создать **паспорт** имперца.

![Static Badge](https://img.shields.io/badge/codex_powered-30%25-green?logo=openaigym)

<div align="center">
  <img src=".github/content/preview.webp" alt="og dsd" />
</div>

## Что это делает

- собирает данные/элементы для паспорта
- отрисовывает визуальный паспорт
- позволяет экспортировать результат в webp

## Стек

- Preact
- Vite
- TypeScript

## Быстрый старт

```bash
npm install
npm run dev
```

Сборка:

```bash
npm run build
```
